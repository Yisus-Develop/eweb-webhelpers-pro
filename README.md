The professional "Swiss Army Knife" for WordPress development. A modular collection of high-performance utilities, security enhancements, and development helpers designed to accelerate your workflow. Part of the **EWEB Plugin Suite**.

## 🚀 Key Features

- **Performance Optimization:** Modular scripts to clean URLs, remove unnecessary WordPress bloat, and optimize assets.
- **Security Hardening:** Built-in helpers to protect the REST API and sensitive endpoints.
- **Developer Tools:** Advanced debugging helpers and ACF utility functions.
- **I18n Ready:** Fully compatible with WPML and multilingual setups.
- **Dynamic Routing:** Includes custom route management for specialized backend logic.

## 🏗️ Architecture

- **`includes/routes.php`**: Centralized route and endpoint management.
- **`includes/class-eweb-github-updater.php`**: Built-in system for seamless updates directly from GitHub.
- **Modular Design**: Load only what you need to keep the site light and fast.

## 🛠️ Installation

1. Upload the `webhelpers` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the **WebHelpers** settings to configure your modular utilities.

## 🔄 Updates

This Pro edition supports automatic updates directly from GitHub. Repository slug: `Yisus-Develop/eweb-webhelpers-pro`.
