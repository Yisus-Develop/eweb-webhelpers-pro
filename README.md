Advanced Backend Utility Suite and API Routing Engine for WordPress. This "Pro" library is designed for developers who need specialized form processing, custom REST endpoints, and dynamic UI components. Part of the **EWEB Plugin Suite**.

## 🚀 Key Features

- **Custom API Routing:** Centralized management for custom endpoints and backend logic via `includes/routes.php`.
- **Advanced Form Processing:** Specialized engine for lead capture and Contact Form 7 integration.
- **Dynamic UI Components:** Custom shortcodes for terminal-style interfaces, member grids, and interactive tabs.
- **Developer Helpers:** Advanced query modifiers for Elementor and ACF utility functions.
- **Multilingual Ready:** Native support for WPML and multi-language deployments.

## 🏗️ Architecture

- **`includes/routes.php`**: The brain of the plugin, managing custom business logic.
- **`includes/helpers/`**: Specialized tools for data manipulation and Elementor queries.
- **`includes/shortcodes/`**: Library of application-specific UI elements.
